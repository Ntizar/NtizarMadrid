# Madrid — Scroll World Landing

Una web landing estilo "scroll-world" que recorre los lugares más icónicos de la ciudad de Madrid.

### Features
- Efecto de "viaje" al hacer scroll.
- Parallax y transiciones cinematográficas.
- Diseño responsive y minimalista.

### Stack
- HTML5
- CSS3 (Variables, Animations, Gradients)
- Vanilla JavaScript (Scroll logic, Keyboard & Touch support)
